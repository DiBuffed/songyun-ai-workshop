# Songyun AI Workshop / 宋韵 AI 工作坊

Generate traditional Chinese Song Dynasty style (宋韵) images using Stable Diffusion.

## Features

- **Songyun LoRA** – Lightweight model for Song Dynasty painting style
- **Preset prompts** – Misty mountains, pavilion, scholar, flower-and-bird, etc.
- **Customizable** – LoRA strength, resolution, guidance scale

## Usage

1. Enter a prompt or select a preset
2. Click **Generate**
3. Download your image

*Note: Free CPU Spaces may be slow (1–3 min per image). GPU upgrade recommended for faster generation.*
